安裝指南

快速開始使用 Pylon Base

安裝步驟

  1. 前置要求

    確保您的伺服器運行 Paper 或其分支（如 Purpur）。建議使用 Minecraft 1.20.1 或更高版本。

  2. 下載 Pylon Core

    Pylon Base 需要 Pylon Core 插件作為基礎。請先從官方網站下載最新版本的 Pylon Core。

  3. 下載 Pylon Base

    從 GitHub Releases 頁面下載最新版本的 Pylon Base JAR 文件。

  4. 安裝插件

    將 Pylon Core 和 Pylon Base 的 JAR 文件放入伺服器的 plugins 資料夾中。

  5. 重啟伺服器

    重啟您的 Minecraft 伺服器以載入插件。

  6. 配置插件

    首次啟動後，插件會在 plugins/PylonBase 資料夾中生成配置文件。您可以根據需要調整配置。

  7. 開始遊戲

    進入遊戲後，使用命令 /pylon 打開插件指南，開始您的 Pylon 之旅！

遊戲內指南

Pylon Base 包含完整的遊戲內指南系統！
使用 /pylon guide 命令可隨時查看所有物品、配方和機器的詳細信息。

配置說明

主配置文件

位於 plugins/PylonBase/config.yml

主要配置項：
  • 語言設定 - 選擇插件語言
  • 研究系統 - 調整研究點數獲取
  • 機器設定 - 每台機器的效率和消耗
  • 流體系統 - 管道流量和溫度限制

配方自訂

位於 plugins/PylonBase/recipes/

可自訂的配方：
  • 研磨配方 - 調整輸入輸出比例
  • 熔煉爐配方 - 修改合金比例和溫度
  • 機器配方 - 自訂處理時間和消耗
  • 魔法配方 - 調整儀式需求
