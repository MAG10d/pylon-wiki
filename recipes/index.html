配方系統

探索 Pylon Base 豐富的配方和製作系統

Pylon Base 包含多種獨特的配方系統，每種都有其特定的用途和機制。

研磨配方

銅粉製作

輸入
  • 銅錠 x1

研磨次數： 2次

輸出
  • 銅粉 x1

核心礦塊處理

輸入
  • 淺層核心礦塊 x1

研磨次數： 6次

隨機輸出
  • 煤炭（權重0.5）
  • 粗銅（權重0.4）
  • 粗錫（權重0.3）

熔煉爐配方

青銅合金

輸入（流體）
  • 熔融銅：88%
  • 熔融錫：12%

溫度： 950°C

輸出
  • 熔融青銅：100%

鋼合金

輸入（流體）
  • 熔融鐵：96%
  • 碳漿料：4%

溫度： 1540°C

輸出
  • 熔融鋼：100%

鐵冶煉（含副產物）

輸入（流體）
  • 粗鐵漿料：100%
  • 碳漿料：50%
  • 硫磺：10%

溫度： 1540°C

輸出
  • 熔融鐵：100%
  • 漿料：40%
  • 熔融鈷：10%
  • 熔融鎳：10%

錘擊配方

金屬板製作

將金屬錠放在對應的方塊上，使用錘子敲擊可製作金屬板。

示例
  • 銅錠 → 銅板
  • 鐵錠 → 鐵板
  • 金錠 → 金板
注意

更高級的錘子有更高的成功率。

魔法祭壇配方

閃光物品製作

在8個基座上放置特定物品，使用催化劑激活祭壇進行儀式。

儀式過程：

  • 將所需物品放置在8個魔法基座上
  • 使用催化劑右鍵點擊中央的魔法祭壇
  • 等待儀式完成
  • 收集製作完成的物品

混合鍋配方

物品混合

將多種物品放入混合鍋中，下方放置火源，右鍵點擊混合。

要點：

  • 普通火源適用於基礎配方
  • 富集地獄岩火焰適用於高級配方
  • 也可作為普通煉藥鍋使用

其他配方類型

壓榨配方

使用壓力機將植物壓榨成植物油，用於液壓系統。

管道彎曲配方

使用液壓管道彎曲機高效製作各種流體管道。

台鋸配方

使用液壓台鋸將原木切成木板，木板切成木棒。

塑形配方

使用磚模對方塊進行塑形，創造特殊形狀。

土窯配方

使用土窯進行原始的高溫冶煉，製作基礎材料。

澆鑄配方

在熔煉爐中將熔融金屬澆鑄成各種形狀的物品。
